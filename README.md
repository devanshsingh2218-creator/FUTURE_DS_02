# FUTURE_DS_02
Customer Retention &amp; Churn Analysis Dashboard using Power BI to identify churn patterns, retention drivers, and customer risk segments.

# Project Overview
This project focuses on analyzing customer retention and churn behavior for a subscription-based business. The objective is to identify churn patterns, key retention drivers, customer lifetime trends, and opportunities to reduce customer loss through data-driven insights.

# Objective
- Analyze customer churn patterns  
- Identify high-risk customer segments  
- Evaluate key retention drivers  
- Understand customer tenure behavior  
- Generate actionable recommendations to improve customer retention

# Tools Used
- Power BI  
- Microsoft Excel  

# Dashboard Features
- Total Customers KPI  
- Churn Customers KPI  
- Churn Rate Analysis  
- Churn by Gender  
- Churn by Contract Type  
- Churn by Monthly Charges  
- Churn by Payment Method  
- Customer Tenure Analysis  
- Risk Segmentation Dashboard  

# Key Insights
- Total Customers: 7,032  
- Churn Customers: 1,869  
- Churn Rate: 27%  
- Month-to-Month contract customers contributed the highest churn  
- Electronic Check users showed higher churn compared to other payment methods  
- Customers with shorter tenure were more likely to churn  
- High-risk customer segment formed a significant portion of the customer base  

# Business Recommendations
- Promote long-term contracts using discounts and loyalty benefits  
- Encourage automatic payment methods to reduce churn  
- Implement proactive retention strategies for high-risk customers  
- Improve onboarding experience for new customers  
- Continuously monitor churn indicators using analytics dashboards
